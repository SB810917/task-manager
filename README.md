# Task Manager

A lightweight personal task manager that runs entirely in the browser — no server, no dependencies, just a single HTML file.

## Features

- **Add tasks** via text input or pressing Enter
- **Complete tasks** with a checkbox (strikethrough styling)
- **Edit tasks** inline by clicking the pencil icon or the task text
- **Delete tasks** individually
- **Persistent storage** — tasks survive page refreshes via `localStorage`
- **Group by date** — tasks are grouped into collapsible folders labelled with the system date (MM/DD/YYYY)
- **Expand / collapse folders** — open/closed state is also saved across refreshes
- **Completion counter** — each folder shows a done/total badge at a glance

## Usage

1. Download or clone this repo
2. Open `tasks.html` in any modern browser
3. Start adding tasks — everything is saved automatically

## No build step required

This app has zero dependencies and requires no installation, bundler, or server. It works offline out of the box.
